# Proof-of-concept Vue.JS for Cannabis Retail.

## Features
* Login Page
* Registration Page
* Store
* Item View
* Simple Setup
* Minimalistic design

[Album featuring pictures of Interface, Hosted on Imgur](https://imgur.com/a/r6Xac9y)

Local Development
==

## Project setup
```
npm install
```

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
